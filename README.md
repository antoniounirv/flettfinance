
# FleetFinance BI

Sistema financeiro para transportadoras com motor de regras avançado:

✅ Caixa vs DRE  
✅ Parcelamento por produto  
✅ Custo já alocado  
✅ Financiamentos  
✅ Estoque com impacto apenas na saída  
✅ Multiempresa (CNPJ)  
✅ Exportação PDF / Excel  
✅ Dashboard gráfico  
✅ Docker + Deploy  

## Rodar local
```bash
cd backend
mvn clean package
docker-compose up --build
```

Frontend:
```bash
cd frontend
abrir index.html
```
